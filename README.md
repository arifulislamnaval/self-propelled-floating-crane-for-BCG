# 📌 Project Overview

| Item | Details |
|------|---------|
| **Project** | 70 Ton Self-Propelled Floating Crane |
| **Client** | Bangladesh Coast Guard |
| **Role** | Project Coordinator & Naval Architect |
| **Classification** | Bureau Veritas (BV) |
| **Delivery** | 11 May 2022 |
| **Status** | ✅ Completed & Delivered |

The **70 Ton Self-Propelled Floating Crane** was developed for the **Bangladesh Coast Guard** to support heavy lifting, rescue and salvage operations, vessel maintenance, and logistics in inland and coastal waters. Constructed under **Bureau Veritas (BV)** classification, the project required multidisciplinary engineering, construction management, and close coordination from design through commissioning.
